# Групповой проект №5. Deep Learning

Проект по курсу «Современные методы анализа данных и машинного обучения» (НИУ ВШЭ, БИ, 2025-26).

## О чем проект

Компания - Myntra, крупный индийский fashion-маркетплейс (аналог Lamoda). Решаем бизнес-задачу автоматизации заведения карточки товара на платформе:

- Рекомендация цены товара по атрибутам карточки (бренд, цвет, категория, рейтинги и т.д.) - полносвязная нейронная сеть на табличных данных. Помогает продавцу сразу поставить адекватную для каталога цену.

## Данные

Датасет - открытая выгрузка каталога Myntra с Kaggle:

- [Myntra Fashion Product Dataset](https://www.kaggle.com/datasets/djagatiya/myntra-fashion-product-dataset) - 14,2 тыс. товаров: цена, бренд, цвет, рейтинги, словарь атрибутов.

Данные в репозитории не храним, они скачиваются автоматически:

```bash
python src/download_data.py
```

## Структура репозитория

```
├── notebooks/            # основные ноутбуки проекта
│   ├── 01_data_eda.ipynb       # постановка задач, получение данных, EDA
│   └── 02_tabular_fcnn.ipynb   # полносвязная сеть: рекомендация цены
├── src/                  # вспомогательные скрипты
│   └── download_data.py        # загрузка данных с Kaggle
├── data/                 # данные (не в гите, качаются скриптом)
├── models/               # локальные веса (не в гите, артефакты лежат в wandb)
├── requirements.txt
└── README.md
```

## Как запустить

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/download_data.py
jupyter notebook notebooks/
```

## Логирование экспериментов

Все эксперименты залогированы в Weights & Biases: параметры моделей и обучения, метрики по эпохам, артефакты обученных моделей, а также обучающие/валидационные данные (артефакты tabular-data) - любой эксперимент воспроизводится имея только доступ к wandb.

Проект в wandb: https://wandb.ai/timoyarrr-tenderscan/gp5-fashion-dl
