# Групповой проект №5. Deep Learning

Проект по курсу «Современные методы анализа данных и машинного обучения» (НИУ ВШЭ, БИ, 2025-26).

## О чем проект

Компания - Myntra, крупный индийский fashion-маркетплейс (аналог Lamoda). Решаем две связанные бизнес-задачи, которые вместе автоматизируют заведение карточки товара на платформе:

1. Рекомендация цены товара по атрибутам карточки (бренд, цвет, категория, рейтинги и т.д.) - полносвязная нейронная сеть на табличных данных. Помогает продавцу сразу поставить адекватную для каталога цену.
2. Определение категории товара по фото - сверточные нейронные сети. Автозаполнение карточки и меньше ошибок в каталоге.

Связка задач: продавец загружает фото и атрибуты -> модель по фото предлагает категорию -> модель по атрибутам рекомендует цену -> продавец подтверждает -> карточка уходит на модерацию и публикуется.

## Данные

Оба датасета - открытые выгрузки каталога Myntra с Kaggle:

- [Myntra Fashion Product Dataset](https://www.kaggle.com/datasets/djagatiya/myntra-fashion-product-dataset) - 14,2 тыс. товаров: цена, бренд, цвет, рейтинги, словарь атрибутов;
- [Fashion Product Images (Small)](https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-small) - 44,4 тыс. товаров с фото 80x60 и атрибутами.

Данные в репозитории не храним, они скачиваются автоматически:

```bash
python src/download_data.py
```

## Структура репозитория

```
├── notebooks/            # основные ноутбуки проекта
│   ├── 01_data_eda.ipynb       # постановка задач, получение данных, EDA
│   ├── 02_tabular_fcnn.ipynb   # полносвязная сеть: рекомендация цены
│   └── 03_images_cnn.ipynb     # CNN: классификация категории по фото
├── src/                  # вспомогательные скрипты
│   └── download_data.py        # загрузка данных с Kaggle
├── data/                 # данные (не в гите, качаются скриптом)
├── models/               # локальные веса (не в гите, артефакты лежат в wandb)
├── requirements.txt
└── README.md
```

## Как запустить

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/download_data.py
jupyter notebook notebooks/
```

## Логирование экспериментов

Все эксперименты залогированы в Weights & Biases: параметры моделей и обучения, метрики по эпохам, артефакты обученных моделей, а также обучающие/валидационные данные (артефакты tabular-data и images-data) - любой эксперимент воспроизводится имея только доступ к wandb.
